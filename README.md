# Audio Flashcards Website — Premium Image Version

This version includes:
- Accent color `#FF6038`
- Minimal mobile-first interface
- Premium image-based visuals (not just icons)
- Header dropdown to jump to any card
- Previous / Next navigation
- Tap card to flip

## Deploy on GitHub Pages
1. Create a GitHub repository.
2. Upload everything in this folder, including the `assets` folder.
3. Go to **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select **main** and **/root**.
6. Open the generated URL on your phone.
